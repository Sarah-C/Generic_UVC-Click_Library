# UVC Click

## Overview

**UVC Click** driver library, based on GUVC-T21GH sensor to achieve UVC light detection in the range of 220nm up to 280nm and light intensity from 0mW/cm² up to 9.3mW/cm², and a MCP3221 ADC converter.
